# Escape The System

A small escape room game built with React, React Router, TypeScript and Context API.

## Start project

npm install
npm run dev

## Implemented requirements

- React Router with routes for /, /rooms/:roomPath and /victory
- Shared inventory with Context API
- Dynamic RoomPage using useParams()
- Puzzle logic where the correct item unlocks the next item
- Solved rooms stay solved through inventory state
- Hint function using useSearchParams()
- Victory page with reset of inventory
